# UHUB
A USB Hub, featuring 2x USB A Ports, but it looks like a USB stick.
I tried 3 variants, but I really liked this one, the other one was with a SD card adapter, and another was with 2x USB-C ports, but both did not fit my form factor that well, so I ended up with this.
BOM in the Excel sheet.
